Textos Grandes e Pequenos

Para deixar um texto menor, usa-se a tag SMALL, em contrapartida, não há tag para deixar um texto maior, a tag BIG é uma tag já obsoleta pois no caso de aumentar um trecho do código o correto seria criar a Formatação do trecho, criar uma classe com CSS no inicio do código.
